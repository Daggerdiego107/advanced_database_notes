## Topics covered
- SELECT queries
- Filtering query results with WHERE
- Sorting query results with ORDER BY

## What I understood
- The SELECT statement is used to query data from a database. It allows you to specify which columns you want to retrieve and from which table.
- The WHERE clause is used to filter records that meet a certain condition. It can be used with various operators such as =, !=, <, >, BETWEEN, LIKE, etc.
- The ORDER BY clause is used to sort the result set in either ascending (ASC) or descending (DESC) order based on one or more columns.
